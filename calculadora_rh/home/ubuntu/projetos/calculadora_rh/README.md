# Calculadora de Custos de RH

Este projeto é uma calculadora web simples para estimar os custos mensais de um funcionário para o departamento de Recursos Humanos. Ele permite inserir o salário base, benefícios e a porcentagem de encargos sociais para obter o custo total.

## Funcionalidades

*   Cálculo do custo total mensal de um funcionário.
*   Entrada de salário base, benefícios e encargos sociais.
*   Interface de usuário intuitiva e responsiva.

## Tecnologias Utilizadas

*   **Frontend:** HTML, CSS, JavaScript

## Como Executar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/calculadora-custos-rh.git
    cd calculadora-custos-rh
    ```
2.  Abra o arquivo `index.html` diretamente no seu navegador web.
